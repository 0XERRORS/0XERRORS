Alias To Make Thing's Fast In Linux

What Is An Alias? Alias Is A Way To Create A Shortcut Or An Alternative Name For A Command Or A Series Of Commands. This Can Be Useful For Simplifying Long Or Complex Commands, Or For Creating Custom Commands That Execute A Series Of Commands In A Specific Order In Linux, an alias is a feature that allows you to create a shortcut or abbreviation for a longer command or series of commands. Aliases can help you save time and increase productivity by reducing the amount of typing you need to do to execute a command.

Here are some examples of how aliases in Linux can help:

1.Shortening frequently used commands:

If you frequently use a long or complex command, you can create an alias for it to make it easier to type and remember. For example, you can create an alias for the command sudo apt-get update and simply type update to execute the command.

2.Creating custom commands:

You can create your own custom commands using aliases. For example, you can create an alias for a series of commands that you frequently use together, such as clean to run commands that clean up your system.

3.Avoiding typos:

Aliases can also help you avoid typos and mistakes when typing commands. For example, if you frequently mistype the ls command as sl, you can create an alias for sl that runs ls instead.

4.Improving readability:

Aliases can make your commands more readable and easier to understand, especially if you create aliases with descriptive names. For example, you can create an alias for a complex command with a long list of options and arguments, and give it a more descriptive name that explains what the command does.

Overall, aliases in Linux can be a powerful tool for improving productivity and making your command line experience more efficient.

So How Can You Use And Add Alias As A Permanent Command?

Depending On Your $SHELL Change Its .rc File Like .bashrc Or .zshrc You Can Use Any Code Editor Like Vim Or Nano For This And In Bottom Just Add Few Lines With Like 

alias youraliasname="command that you choose"

In youraliasname Set What Alias/Command You Like And In Double Quote "command that you choose" After That Save The File And Use One Command To Activate The .rc File It Can Be .bash Or .zshrc File Depending On Your $SHELL You Have Use source .zshrc or source .bashrc In Your Terminal Then You Can Use Your Alias

Thats It Now You Can Use Alias Heres Are My Few Alias That I Use

#extra alias

             
          alias u="sudo apt update && sudo apt upgrade"
          alias add="sudo apt install" $""
          alias remove="sudo apt purge && sudo apt remove" $""
          alias dpkgc="sudo dpkg --configure -a"
          alias r="sudo rm -r" $""
          alias gitc="sudo git clone" $""
